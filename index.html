Projects

PlanMyJourney – Travel Request Web Application

PlanMyJourney is a web-based travel request system that allows users to register or log in and submit trip requests by selecting from multiple predefined travel packages. Each package dynamically displays its details and calculates the total cost based on group size. Admins (owners) can log in separately to view, edit, delete, and print all user-submitted records. The application ensures real-time interaction using JavaScript and securely handles data using Java Servlets and Hibernate with a MySQL database.

Key Features:

  • User and Admin login systems.
  • Responsive trip request form with dynamic package display.
  • Real-time cost calculations based on selected package and group size.
  • Record display with pagination and print functionality.
  • Built using Java (JSP/Servlets), Hibernate, MySQL, HTML/CSS, Bootstrap, and JavaScript.

Tech Stack:

HTML, CSS, Bootstrap, JavaScript, JSP, Servlets, JDBC, Hibernate, MySQL, MVC Architecture
